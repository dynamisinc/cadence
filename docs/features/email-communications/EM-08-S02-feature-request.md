# Story: EM-08-S02 - Feature Request Submission

**As a** User,  
**I want** to submit feature requests,  
**So that** I can suggest improvements that would help my workflow.

## Context

Feature requests help prioritize product development based on user needs. A simple submission form encourages feedback.

## Acceptance Criteria

- [ ] **Given** I'm logged in, **when** I click "Request Feature", **then** I see feature request form
- [ ] **Given** form, **when** filling out, **then** I provide: title, description, use case
- [ ] **Given** form submitted, **when** saved, **then** email sent to product feedback address
- [ ] **Given** submission, **when** successful, **then** I see "Thank you for your feedback"

## Out of Scope

- Feature voting/ranking
- Public roadmap visibility

## Dependencies

- EM-01-S01: ACS Email Configuration

## Effort Estimate

**2 story points** - Similar to bug report, simpler form
